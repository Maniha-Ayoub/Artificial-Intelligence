# Quizora - AI-Powered Examination & Learning Platform
Quizora is an intelligent, full-stack web application built to facilitate both adaptive learning and strictly proctored examinations. By leveraging Generative AI and modern web technologies, Quizora provides a dynamic educational experience tailored to individual student needs while ensuring the absolute integrity of formal assessments.

# Key Features
# 🎓 Adaptive Learning Mode
- Dynamic Questions: AI generates unique questions on the fly based on selected categories.
- Adaptive Difficulty: The system automatically scales difficulty up or down based on your previous answers.
- Smart Assistance: Request AI-generated hints and detailed explanations.
- PDF to Quiz: Upload study material (PDFs) and let the AI automatically generate a relevant quiz!
# 🛡️ Formal Examination & Proctoring
- Strict Anti-Cheating: Features tab-switching detection, fullscreen enforcement, and active webcam monitoring.
- Input Restrictions: Right-clicking and copy-pasting are disabled.
- Three-Strike Rule: Suspicious behavior triggers warnings. Maximum warnings result in automatic exam termination.
# 🎮 Gamification
- Streaks & Points: Earn points and maintain daily streaks to compete.
- Achievements: Unlock badges like 'Quick Learner', 'Steady Scholar', and 'Master Mind'.
- Global Leaderboard: Compete with other students for the top spot.
# 👥 Role-Based Access
- Student Portal: Access learning modules, take exams, and view personalized stats.
- Admin Dashboard: Manage exams, create/delete questions, and view aggregated student performance records.
# Technology Stack
- Backend: Python, Flask
- Database: SQLite
- Frontend: HTML5, Vanilla JS, CSS (Glassmorphism & modern UI)
- AI Integration: Generative AI models for questions/feedback, face-api.js for webcam proctoring, pypdf for document parsing.
